# 👾 Nexus Chatbot

Ein smarter, leichtgewichtiger Chatbot, der komplett in **JavaScript (Node.js)** geschrieben ist. Nexus läuft direkt im Terminal und wurde als lernfähiges System entwickelt, das über eine einfache **JSON-Datenbank** trainiert werden kann.

---

## ✨ Highlights & Features

- **🧠 Trainierbares Gedächtnis:** Der Bot liest alle seine Antworten aus einer `training-data.json`. Du kannst die Datei im Editor erweitern, und Nexus weiß sofort mehr – ganz ohne den Code ändern zu müssen!
- **💻 Minimalistisches Terminal Interface:** Keine unnötige Grafik, kein Extras. Pure, schnelle Performance direkt in deiner Konsole.

---

## 🛠️ Technische Details (Wie es funktioniert)

Nexus nutzt die Standard-Module von Node.js:
- `fs` (File System), um die Trainingsdaten blitzschnell zu laden und zu parsen.
- Eine logische Abfrage, die die Eingabe des Nutzers direkt mit den gespeicherten Fragen in der JSON-Datenbank abgleicht.
